# Learner TOB
This plugin was created for Rancour PVM OSRS Clan. It is intended to help learner raiders understand the basics of TOB.
Right now, the only functionality is a smart MDPS equipment setup helper. Based off the gear in your bank, it will suggest 
the best equipment and inventory setup. This will be expanded to RDPS, SFRZ, and NFRZ. 
After that room by room features will be added.
